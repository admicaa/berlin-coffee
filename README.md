# Berlin Coffee center
>find coffee shops in Berlin
## Prerequisites
* nodejs
* npm
* react
## Dependencies
* **reactjs**
* **react-bootstrap**
* **react-dom**
* **google-maps-react**
* **recompose**
# Installing
```
npm install 
```
## development Mode
```
npm start
```
### production mode
```
npm run build 
```
